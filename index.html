Como estruturar de forma básica o HTML?

A estrutura básica de HTML é compsota pelas "tags" e Instrução

  1. !DOCTYPE
    Essa é a instrução que define a versão do HTML usado no documento.
  2. HTML
    Abre a página.Nenhuma marcação deve ocorrer após a tag de fechamento.
  3. Head
    4. Abre a seção principal, que não aparece na janela do navegador, mas contém principalmente informações sobre o documento HTML chamado de Metadados.
  4. Body
    5. Abre a parte do documento exibida aos usuários, ou seja todo o conteúdo visível ou audível de uma página. Nenhum conteúdo deve ser adicionado após a tag de fechamento.

Falando sobre Tags

Todas as marcações são realizadas pelas Tags.

Os navegadores interpretam essas instruções e a partir disso renderizam uma página.

A forma mais comum de utilização das Tags são como delimitadores de estilo ou de conteúdo, ou seja ela agrupa os conteúdos de acordo com o tipo, forma e formatações e com isso outras lingagens comp por exemplo CSS podem aplicar estilizações a este conteúdo

(Voltar)

Atributos Básicos

Atributos são informações que acrescentamos nas Tags para que elas se comporte de maneira esperada.

Alguns exemplos de atributos são: acesskey, class, draggable, hidden, id, lang, style, tabindex e title.

(Voltar)

Formatação de alguns textos

Texto é tudo aquilo que os usuários vão ler na página. No caso a Tipografia

As principais Tags para formatação de um texto são h1 a h6, p, br, blockquote, strong, i, u, sub, sup entre outras.

(Voltar)

Diferença entre listas ordenadas e não ordenadas

Uma lista Ordenada é uma lista de itens. As "Lista de itens" são marcadas como números.

Uma lista ordenada começa com a Tag < Ol >. Cada item da lista começa com a Tag < li > que segue uma sequência de 1, 2, 3 por diante.

Uma lista Não ordenada é uma lista de itens. As "Lista de itens" são marcadas com bullets (Tipicamente pequenos círculos pretos) e começa com uma Tag < ul >
